Galaxy is a web based data analysis platform. It is easy to use, and complete free. Galaxy offers over eight thousand analysis tools. Galaxy is widely used, it currently has over ten thousand publications. --- # Why use Galaxy? - It is **easy**! - No installation, all you need is a browser. - No complex commands, just point and click! - **Cross-domain**: bioinformatics, chemistry, ecology, climate science, .. ![Galaxy resource banner]({{site.baseurl}}/topics/introduction/images/titleless-use-resource-banner.png) ??? Data analysis can be very complex. It often requires specialized programming knowledge or command line skills. Galaxy makes this a lot easier by providing a graphical user interface to popular data analysis tools. Galaxy is web based. This means you don't have to install anything, just open your browser and go to Galaxy. Galaxy supports many different scientific domains. --- # How do I use Galaxy? --- # Find a Galaxy server - **The Big Three**: Galaxy Main ([usegalaxy.org](https://usegalaxy.org)), Galaxy Europe ([usegalaxy.eu](https://usegalaxy.eu)), Galaxy Australia ([usegalaxy.org.au](https://usegalaxy.org.au)) .image-75[![usegalaxy.* servers]({{site.baseurl}}/shared/images/usegalaxy_servers.png)] - Many other smaller domain-specific Galaxies available (125+) - List of all Galaxies: https://galaxyproject.org/use/ ??? There are three main Galaxy servers. Galaxy Main, Galaxy Europe, and Galaxy Australia. These three Galaxies have the biggest teams behind them, and offer the most tools and resources. You can register an account for free on any of these servers. In addition, there are many smaller galaxy servers to choose from. Many of these are domain specific. For example, Galaxy proteomics focuses on proteomics tools and workflows. --- # The Galaxy Interface - Three main panels - **Left:** Available Tools - **Middle:** View your data and run tools - **Right:** Full record of your analysis **history** ![Galaxy user interface]({{site.baseurl}}/topics/introduction/images/galaxy_interface.png) --- # Uploading data .pull-left[ - Upload from **your computer** - Import files **from URL** - Import from **public data stores** - UCSC, NCBI, ENA, many more.. ] .pull-right[ ![Galaxy upload button]({{site.baseurl}}/topics/introduction/images/upload-data.png) ] --- # History .pull-left[ - Files appear in your **history** - Three buttons - {% icon galaxy-eye %} **View** the file - {% icon galaxy-pencil %} **Edit attributes** - e.g. change name - {% icon galaxy-cross %} **Delete file** - **Click** to expand - more info and options ] .pull-right[ ![expanding a dataset]({{site.baseurl}}/topics/introduction/images/galaxy_history_dataset_expand.gif) ] --- # Finding a tool .pull-left[ - Explore tool panel sections - Enter name in **Tool search** box ] .pull-right[ ![tool search box]({{site.baseurl}}/topics/introduction/images/tool_search.png) ]
